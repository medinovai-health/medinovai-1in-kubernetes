# Heartbeat — Clinical

## Check Frequency: Every 30 minutes

### Verification Checks
1. **Test Suite**: All tests passing (unit, integration, compliance)
2. **Critical Vulnerabilities**: None unmitigated (fail on high/critical)
3. **FHIR Validation**: Schema validation passing on representative resources
4. **PHI Detection Rules**: Active and correctly configured; no PHI in logs or embeddings
5. **Audit Trail**: Gap-free write continuity; no missing time windows
