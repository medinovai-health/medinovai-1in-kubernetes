## Contributing

### Branch Naming

All branches should be named using the following convention:

`<type>/<bmad-task-id>/<short-description>`

- **type**: feat, fix, chore, etc.
- **bmad-task-id**: The ID of the BMAD task.
- **short-description**: A short description of the change.

### Conventional Commits

All commit messages should follow the Conventional Commits specification.

### Code Review

All pull requests must be reviewed and approved by at least two members of the @myonsite-healthcare/medinovai-core-team before being merged.
