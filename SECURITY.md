# Security Policy

## Reporting a Vulnerability

The MedinovAI team and MyOnsite Healthcare take security vulnerabilities very seriously. We appreciate your efforts to responsibly disclose your findings, and will make every effort to acknowledge your contributions.

To report a security vulnerability, please email [security@medinovai.com](mailto:security@medinovai.com) and include the repository name, a description of the vulnerability, and steps to reproduce it. We will respond to your report within 48 hours.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Compliance and Standards

We are committed to upholding the highest standards of security and compliance. Our services are designed to be compliant with the following standards:

- **HIPAA:** The Health Insurance Portability and Accountability Act
- **GDPR:** The General Data Protection Regulation
- **OWASP:** The Open Web Application Security Project
- **SOC 2:** Service Organization Control 2
