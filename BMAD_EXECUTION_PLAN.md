### **Core Requirements:**
1. **Configurable Chatbots:** Every UI must have a trainable chatbot
2. **Model Configuration:** Model names must be configurable per UI/task
3. **Smallest Suitable Models:** Use optimal model size for each specific task
4. **Adaptive Learning:** Agents must learn and improve from interactions
5. **3-Step Guidance:** Every response must offer next 3 steps to user
